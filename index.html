O que é Install Fest?

Um Installfest é um evento, geralmente patrocinado por um Grupo de Usuários Linux ou universidade, em que pessoas se reúnem para realizar instalações em massa de sistemas operacionais de computador ou software, principalmente o sistema operacional Linux e outros softwares de código aberto. Mesmo que você não esteja pronto para instalar o Linux em sua própria máquina, você poderá participar e conhecer um pouco mais do mundo Linux.

O que é Linux?

Linux é um sistema operacional alternativo ao Windows. Ele é estável, seguro e confiável. O Linux tem várias aplicações gráficas disponíveis, incluindo jogos, processadores de texto, mp3 players, editor de imagens, web browsers entre outros, podendo ser utilizados dependendo da capacidade de sua máquina. Possui também uma grande seleção de softwares para servidores, incluindo servidores web, sistemas de banco de dados, linguagens de script e ferramentas para administração remota.

Eu já tenho Windows/Mac instalado na minha máquina, posso ter o Linux também?

Sim. Os dois sistemas podem ser instalados na mesma máquina, porém em partições diferentes do disco. Isso é o que chamamos de dual boot. Mas não faremos a instalação de Windows ou qualquer outro Aplicativo Proprietário durante o Installfest.

Quanto tempo demora a instalação?

Isso vai depender muito do seu computador. Num computador menos potente, com certeza, irá demorar muito mais que em um computador mais potente. A instalação tanto pode durar 40 minutos como 2 horas ou mais. É muito difícil definir isso.

O que devo fazer para me preparar para o Install Fest?

- Faça backup dos seus dados. Por precaução, você deve fazer uma cópia dos seus dados importantes no computador do seu amigo ou em CD-ROM, não nos responsabilizamos por dados perdidos.

O que devo levar para o Install Fest?

- Seu notebook. Sem ele não será possível participar da Install Fest.
- Não esqueça o pen drive. Por precaução, leve um pen drive vazio de no mínino 8GB para qualquer imprevisto (não é obrigatório).

Como vão instalar o linux no meu computador?

A partir de um CD de instalação do Linux ou Pendrives.

Que softwares estarão disponíveis durante o Installfest?

Muitos iniciantes imaginam que se trata somente de instalação de softwares para Linux ou mesmo o próprio Linux, mas um Installfest possui uma extensa lista de softwares livres equivalentes aos softwares proprietários para serem utilizados no Windows que poderão ser instalados no dia.
